#GUESS-THE-NUMBER (1-20)

## Данный проект представляет собой игру в стиле "угадай число".

Реализовано:

- Сама игра
- Тёмная тема
- Ауидо плеер

Используемые инструменты \ технологии:

1. [Bun](<[text](https://bun.com/)>)
2. [Vite](<[text](https://vite.dev/)>)
3. [TypeScript](https://www.typescriptlang.org/)
4. [SCSS](https://sass-lang.com/)

После скачивания репозитория нужно создать папку /public/audio и закинуть туда песни
